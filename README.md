# GADSLeaderBoard
GADs Practice Project to show learners leaderboard

## Libraries used
* Retrofit - To interact with Api End point
* Moshi - To Deserialize and map Json Models
* Coil - To load images
* Coroutine - For Asyncronous data processing
* Koin - For Dependency Injection
* ViewModel
* LiveData
